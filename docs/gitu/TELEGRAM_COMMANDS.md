# Gitu Telegram Bot Guide

Interact with Gitu directly from Telegram to manage your tasks, notebooks, and research on the go.

## Getting Started
1. Search for the Gitu Bot on Telegram (e.g., `@GituBot`).
2. Start a chat and tap **Start**.
3. Link your account if prompted (follow instructions in the NotebookLLM app).

## Available Commands

| Command | Description |
|---------|-------------|
| `/start` | Initialize the bot and see a welcome message. |
| `/help` | Show the list of available commands and usage tips. |
| `/status` | Check your Gitu connection status, message stats, and active sessions. |
| `/notebooks` | List your accessible notebooks. |
| `/session` | View detailed information about your current chat session. |
| `/clear` | Clear the current conversation context/history. |
| `/settings` | View your current user settings. |

## Troubleshooting
- **Account Not Linked**: If the bot says you aren't linked, go to the NotebookLLM app settings and verify your Telegram integration.
- **No Response**: Try running `/clear` to reset the session if the bot gets stuck.
